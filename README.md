SenchaCouch
===========

SenchaCouch is a small library for ExtJS and Sencha Touch for performing CRUD operations and writing nested data sets to
a CouchDB compatible database.

CouchDB Compatible databases(that i know of):

- CouchDB (duh)
- Cloudant
- PouchDB
- Couchbase Lite
